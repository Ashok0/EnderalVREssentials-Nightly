
Release date: October 8th, 2021

Updated Tools:
- DynDOLOD Standalone 3.0 Alpha-47: DynDOLOD is a set of simple tools based on xEdit/xLODGen to automatically create a Skyrim mod based on the load order which adds distant LOD for objects and trees to Skyrim. By combining DynDOLOD with xLODGen users can create drastically enhanced static object LOD + tree LOD and the new, optional dynamic distant object LOD in a few simple steps.

New Mods:
- Forgotten Dungeons - Palm Cove v1.0: Forgotten Dungeons implements cut content, originally made by SureAI, with updates and fixes to fit seamlessly into the game.

Updated Mods:
- DynDOLOD Resources SE v3.00 Alpha-13: DynDOLOD is a set of simple tools based on xEdit/xLODGen to automatically create a Skyrim mod based on the load order which adds distant LOD for objects and trees to Skyrim. By combining DynDOLOD with xLODGen users can create drastically enhanced static object LOD + tree LOD and the new, optional dynamic distant object LOD in a few simple steps.
- Sprint Jump VR v1.1.2: Fixed a bug that caused a momentary pause 1 second after landing from a sprint jump. Fixed a bug that can potentially cause a freeze/ctd, some other optimizations. Fixed a bug that caused super speed after landing from sprint jumping when weapons are sheathed.
- VR FPS Stabilizer v1.3.2: Please see Nexus changelog for updates.  Mod is disabled by default in this Wabbajack release and is only used for its console features to disable TAA FX.
- Embers XD v2.4.0b: Custom inn fire pit logs. Dynamic campsite lights. Makeover for oven. Campsite collisions.  Improved giant campsite blending. Fixed light color of wall torches. Other bits and pieces.
- Diverse Weather v1.11: Reduced volumetric lightning in fog weathers.
- Enderal SE - True Storms v1.01: Fixed the volumetric lightning with the heavy fog weathers.
